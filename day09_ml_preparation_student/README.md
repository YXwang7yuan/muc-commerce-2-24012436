## 用自己的话回答

- 什么是特征，什么是标签：特征是用来做判断的各类自变量数据（比如使用时长、登录频率等）；
                        标签是想要预测的目标变量，本例中就是用户是否流失 Churn。
- 为什么要保留测试集：测试集不参与模型训练，用来模拟真实业务环境，客观检验模型在陌生数据上的泛化能力，
                    防止模型只是记住训练数  据造成过拟合。
- 为什么83%准确率仍可能没有用：比如流失用户占比很低，预测全部用户不流失也能拿到较高准确率。
                             所以准确率无法体现能否识别少数流失人群，流失客户依旧可能出现漏判。
